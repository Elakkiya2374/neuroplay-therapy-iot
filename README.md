# NeuroPlay Therapy

## Overview

NeuroPlay Therapy is an IoT-based system designed for the early detection of developmental disorders through therapeutic play activities.

## Objective

The project aims to monitor children's interactions using sensors and collect real-time data that can help identify behavioral patterns at an early stage.

## Technologies Used

* Touch Sensors
* Motion Sensors
* Firebase
* ThingSpeak
* Google Sheets
* IoT Platforms

## Features

* Real-time sensor monitoring
* Data collection and cloud storage
* Behavioral activity tracking
* Automated data logging
* Early pattern observation
